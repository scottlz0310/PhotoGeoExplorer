要約: PhotoGeoExplorer はプライバシーを重視したデスクトップアプリです。データは端末内で処理され、個人情報を収集・保存・共有しません。

1. はじめに

PhotoGeoExplorer（以下「本アプリ」）は、写真の位置情報を地図上で表示・探索する Windows デスクトップアプリです。本ポリシーでは、本アプリがどのようにデータを取り扱い、プライバシーを保護するかを説明します。

2. 収集しない情報

本アプリはプライバシーを重視して設計されています。本アプリは次のことを行いません。

  • 個人情報の収集
  • 利用状況や行動のトラッキング
  • アカウント登録・ログインの要求
  • 写真やデータのサーバーへのアップロード
  • 第三者への個人情報の共有

3. 端末内でのデータ処理

本アプリのデータ処理は、すべてお使いの端末内で完結します。

  • 写真ファイル: 本アプリは、あなたが明示的に選択したフォルダー内の写真のみを読み取ります。写真は端末内に留まり、アップロードされません。
  • EXIF メタデータ: 位置情報表示のために、写真内の EXIF/GPS データを読み取ります。処理は端末内で行われ、外部へ送信されません。
  • 設定: 言語・テーマなどの設定は、端末内の %LocalAppData%\PhotoGeoExplorer\settings.json に保存されます。
  • ログ: トラブルシュート目的で、端末内の %LocalAppData%\PhotoGeoExplorer\Logs\app.log にアプリログが保存されます。
  • 地図タイルキャッシュ: 地図表示のために取得したタイルは、性能向上と通信量削減のため %LocalAppData%\PhotoGeoExplorer\Cache\Tiles にキャッシュされます。

4. インターネット通信

本アプリは、次の目的に限って最小限のインターネット通信を行います。

  • 地図タイル: 地図表示のため、OpenStreetMap または Esri WorldImagery のサーバーから地図タイルを取得します。これらのリクエストに個人情報は含めません。
  • 更新チェック: 新バージョン確認のため、GitHub Releases を参照します。これらのリクエストに個人情報や写真データは含めません。

初回の地図タイルがキャッシュされた後は、オフラインでも利用できます（地図の新規取得は除く）。

5. 第三者サービス

本アプリは次の第三者サービスを利用しますが、個人情報を共有しません。

  • OpenStreetMap: 地図タイルを OpenStreetMap のタイルサーバーから取得します。詳しくは Privacy Policy を参照してください。
  • Esri WorldImagery: 衛星画像タイルを Esri のサーバーから取得します。詳しくは Privacy Policy を参照してください。
  • GitHub: 更新チェックは GitHub API 経由で行われます。詳しくは GitHub Privacy Statement を参照してください。

6. Google Maps 連携

本アプリには、写真の位置をブラウザーで Google Maps を開く任意機能があります。「Google Maps で開く」をクリックした場合、緯度・経度がブラウザーの Google Maps に渡されます。この操作はあなたが明示的に行う場合にのみ実行されます。詳しくは Google のプライバシーポリシー を参照してください。

7. データの安全性

データは端末内に保存されるため、端末のセキュリティ管理は基本的にユーザーの責任となります。

  • Windows とセキュリティソフトを最新の状態に保つ
  • 写真フォルダーのアクセス権限を適切に管理する
  • 重要な写真は定期的にバックアップする

8. 子どものプライバシー

本アプリは、子どもから個人情報を意図的に収集することはありません。本アプリは一般向けであり、アカウント登録等も不要です。

9. 本ポリシーの変更

本ポリシーは必要に応じて更新される場合があります。変更がある場合は本ページに掲載し、「最終更新日」を更新します。変更後に本アプリを継続して利用することで、更新後の内容に同意したものとみなされます。

Summary: PhotoGeoExplorer is a privacy-focused desktop app. Your data is processed locally and no personal information is collected, stored, or shared.

1. Introduction

PhotoGeoExplorer (the "App") is a Windows desktop application that displays and explores photo location information on a map. This policy explains how the App handles data and protects your privacy.

2. Information We Do Not Collect

The App is designed with privacy in mind. We do not:

  • Collect personal information
  • Track usage or behavior
  • Require account creation or sign-in
  • Upload photos or data to a server
  • Share personal information with third parties

3. Local Data Processing

All data processing is performed locally on your device.

  • Photo files: The App reads only photos in folders you explicitly select. Photos remain on your device and are never uploaded.
  • EXIF metadata: EXIF/GPS data is read to display locations. Processing is local and not transmitted externally.
  • Settings: Language/theme settings are stored at %LocalAppData%\PhotoGeoExplorer\settings.json.
  • Logs: For troubleshooting, logs are stored at %LocalAppData%\PhotoGeoExplorer\Logs\app.log.
  • Map tile cache: Tiles are cached at %LocalAppData%\PhotoGeoExplorer\Cache\Tiles to improve performance and reduce network traffic.

4. Network Communication

The App performs minimal network access only for the purposes below:

  • Map tiles: Map tiles are fetched from OpenStreetMap or Esri WorldImagery servers. These requests do not include personal information.
  • Update checks: GitHub Releases are queried for new versions. These requests do not include personal information or photo data.

Once map tiles are cached, the App can be used offline (except for fetching new tiles).

5. Third-Party Services

The App uses the following services without sharing personal information:

  • OpenStreetMap: Map tiles are fetched from OpenStreetMap. See their Privacy Policy.
  • Esri WorldImagery: Satellite tiles are fetched from Esri. See their Privacy Policy.
  • GitHub: Update checks use GitHub APIs. See the GitHub Privacy Statement.

6. Google Maps Integration

The App includes an optional feature to open photo locations in Google Maps. When you click "Open in Google Maps", latitude and longitude are passed to Google Maps in your browser. This occurs only when you explicitly choose to do so. See Google's Privacy Policy.

7. Data Security

Because all data is stored locally, device security is the user's responsibility.

  • Keep Windows and security software up to date
  • Manage access permissions for your photo folders
  • Back up important photos regularly

8. Children's Privacy

The App does not knowingly collect personal information from children. It is intended for general audiences and does not require accounts.

9. Policy Changes

This policy may be updated as needed. When changes occur, we will update the "Last updated" date on this page. Continuing to use the App after changes indicates acceptance of the updated policy.
