# リリースノート（v1.4.0）

## 概要

フォルダ操作やファイル関連付けなど、日常の使い勝手を改善しました。地図とプレビューの連携を強化し、サムネイルの品質も向上しています。

## 主な追加

- .jpg/.jpeg/.png/.heic のファイル関連付けとファイル起動対応
- パンくずの区切り「>」クリックで子階層を開く操作
- 地図ピンクリックで該当写真へフォーカス

## 主な変更

- フォルダ移動をダブルクリック操作に変更
- プレビュー最大化時の地図エリア調整と高 DPI 画面フィットの改善
- マップピン位置を先端基準に調整
- サムネイル生成の高品質化
- タスクバー背景の透明化用 unplated アイコン追加
- タイトルバーのアイコン反映

## 修正

- 位置情報なし写真で Null Island にピンが立つ問題を回避

## 削除

- Store 方針に合わせて更新確認 UI を削除

## WACK 結果

- Required: 0
- Optional: [88] ブロック済みの実行可能ファイル
