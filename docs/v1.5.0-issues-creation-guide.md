# v1.5.0 Issue作成ガイド

このドキュメントは、v1.5.0に向けた8件のIssueを作成するための詳細ガイドです。

## 前提条件

- GitHub CLI (`gh`) がインストールされていること
- リポジトリへのアクセス権限があること
- v1.5.0 マイルストーンが作成されていること（なければ作成してください）

## マイルストーンの確認・作成

```bash
# 既存のマイルストーンを確認
gh milestone list

# v1.5.0 マイルストーンがなければ作成
gh milestone create v1.5.0 --title "v1.5.0" --description "次期メジャーリリース"
```

## 一括実行スクリプト

上記のコマンドを一括で実行したい場合は、同梱の `create-v1.5.0-issues.sh` スクリプトを使用してください。

```bash
chmod +x scripts/create-v1.5.0-issues.sh
./scripts/create-v1.5.0-issues.sh
```

## 注意事項

1. スクリプト実行前に、必ずGitHub CLIで認証を行ってください：
   ```bash
   gh auth login
   ```

2. v1.5.0 マイルストーンが存在しない場合は、事前に作成してください。

3. ラベルが存在しない場合は、自動的に作成されます。

4. 各Issueの作成後、Issue番号が表示されます。必要に応じてメモしてください。

## 作成後の確認

すべてのIssueが作成されたら、以下のコマンドで確認できます：

```bash
# v1.5.0 マイルストーンのIssue一覧を表示
gh issue list --milestone v1.5.0

# scottlz0310 にアサインされたIssue一覧を表示
gh issue list --assignee scottlz0310
```
